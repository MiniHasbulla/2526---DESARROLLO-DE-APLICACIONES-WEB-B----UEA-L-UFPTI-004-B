Grid Bootstrap

Implementación de grid responsivo con 3 columnas (col-md-4) usando el sistema de grid de Bootstrap 5.

Características implementadas:
  • Sistema grid: row + col-md-4
  • Cards con sombras y bordes redondeados
  • Imágenes responsivas con img-fluid
  • Componente collapse de Bootstrap

CSS3 Avanzado

Uso de CSS3 con animaciones, transforms, transitions, media queries y estilos personalizados que sobrescriben Bootstrap.

Técnicas CSS3 usadas:
  • Animaciones con @keyframes
  • Transforms y transitions en hover
  • Media queries personalizadas
  • Variables CSS personalizadas
  • Gradientes y sombras avanzadas
