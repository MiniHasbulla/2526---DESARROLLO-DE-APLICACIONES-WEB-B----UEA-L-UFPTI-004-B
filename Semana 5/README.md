# Galería de Imágenes Interactiva

Una galería de imágenes interactiva creada con HTML, CSS y JavaScript puro que permite agregar, seleccionar y eliminar imágenes de manera dinámica.

## Características

- **Agregar imágenes**: A través de URLs de imágenes
- **Seleccionar imágenes**: Haciendo clic en cualquier imagen de la galería
- **Eliminar imágenes**: Elimina la imagen seleccionada
- **Vista previa**: Muestra una vista ampliada de la imagen seleccionada
- **Diseño responsive**: Se adapta a diferentes tamaños de pantalla
- **Contador de imágenes**: Muestra cuántas imágenes hay en la galería
